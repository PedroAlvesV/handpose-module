# HandPose Module

Hand pose recognition module with MQTT publishing.

## Installation

```bash
$ git clone https://github.com/PedroAlvesV/handpose-module.git
$ cd handpose-module 
$ pip install -r requirements.txt
```

## Running

### Standalone version

```bash
$ git checkout standalone
$ python handpose.py -d 2
```
